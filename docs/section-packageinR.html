3.1 Installation

If you do not have R installed on your computer, please go to Appendix 1. Once you have R installed, you can install the framinghamPredictor package as per the instructions below:

Windows 7 or Later

  1. Download and Install the latest version of R from https://cran.r-project.org/bin/windows/base/

  2. Download and Install R Studio from https://www.rstudio.com/products/rstudio/download/

  3. Using either an R session in Terminal or in R Studio, install the package devtools:

install.packages('devtools')
  1. Install framinghamPredictor from GitHub:

Note: This package has been set to private.

devtools::install_github('resplab/framinghamPredictor')

Mac OS Sierra and Later

  1. Download and Install the latest version of R from https://cran.r-project.org/bin/macosx/

  2. Download and Install R Studio from https://www.rstudio.com/products/rstudio/download/

  3. Using either an R session in Terminal or in R Studio, install the package devtools:

install.packages('devtools')
  1. Install epicR from GitHub:
devtools::install_github('resplab/framinghamPredictor')

Ubuntu 16.04 and Later

  1. Install R if you donâ€™t have it installed already. See Appendix 1 for installation guide.

  2. Download and Install R Studio from https://www.rstudio.com/products/rstudio/download/

  3. Using either an R session in Terminal or in R Studio, install the package devtools:

install.packages ('devtools')
  1. Install framinghamPredictor from GitHub:
devtools::install_github('resplab/framinghamPredictor')
install.packages("devtools")

3.2 Running the Model

Now that you have installed framinghamPredictor, you can load the library:

library(framinghamPredictor)

There should be the default input predictors which come preloaded:

print(sample_predictors)
>   fev1_0 fvc_0 age triglycerides hematocrit albumin globulin ALP WBC QRS_intv wine beer cocktail height smoke_year
> 1      1     1  50            NA         NA      NA       NA  NA  NA       NA   NA   NA       NA    180         20
>   daily_cigs    sex broncho dyspnea_exc night_sym
> 1          4 female

To run the model, you can use these sample_predictors, or adjust them yourself:

make_predictions(resp_var="fev1", predictors=sample_predictors)
