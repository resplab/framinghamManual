3 Using the Package in R

3.1 Installation

If you do not have R installed on your computer, please go to Appendix 1. Once you have R installed, you can install the framinghamPredictor package as per the instructions below:

Windows 7 or Later

  1. Download and Install the latest version of R from https://cran.r-project.org/bin/windows/base/

  2. Download and Install R Studio from https://www.rstudio.com/products/rstudio/download/

  3. Using either an R session in Terminal or in R Studio, install the package devtools:

install.packages('devtools')
  1. Install framinghamPredictor from GitHub:
devtools::install_github('resplab/framinghamPredictor')

Mac OS Sierra and Later

  1. Download and Install the latest version of R from https://cran.r-project.org/bin/macosx/

  2. Download and Install R Studio from https://www.rstudio.com/products/rstudio/download/

  3. Using either an R session in Terminal or in R Studio, install the package devtools:

install.packages('devtools')
  1. Install epicR from GitHub:
devtools::install_github('resplab/epicR')
